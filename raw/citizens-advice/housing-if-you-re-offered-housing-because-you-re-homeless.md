#  If you're offered housing because you're homeless

###  If you’ve been offered a home with a private landlord

The tenancy must be a ‘rolling tenancy’. This means the tenancy continues monthly or weekly with no set end date.

If you’re thinking of refusing an offer, you should [talk to an adviser](/about-us/contact-us/contact-us/contact-us/).

Find out more about [challenging a housing offer](/housing/homeless-help/homelessness/challenging-your-homeless-decision/if-you-disagree-with-your-homeless-application-decision/#h-if-you-think-the-housing-you-re-offered-isn-t-suitable).
