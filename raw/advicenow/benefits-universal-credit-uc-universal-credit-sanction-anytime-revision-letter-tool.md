#  Universal Credit sanction anytime revision letter tool

Been given a DWP sanction? Don’t just accept it. This free tool makes it easy to challenge a Universal Credit (UC) sanction even if more than 1 month has passed. Almost all UC sanctions can be overturned.

[Quality checked](/get-help/why-you-can-trust-our-content)

England, Wales and Scotland only

February 2026

This free tool will write you a letter that explains why you should not have been given a sanction and asks for the decision to be revised.

It does not matter how long ago you were given the sanction. You can challenge it now, using this tool to ask for an anytime revision.



Almost all sanctions can be successfully challenged.

Even if you have already explained why you had good reason for the error, it is still worth asking for a revision and (if necessary) appeal.

## How to use this tool

Log in to use this letter tool to explain again why you should not have been given a sanction. You can save your progress and come back to it if you need to.

Why challenge a sanction?

  * Every sanction should be challenged.
  * Most sanctions are overturned if challenged.
  * If your sanction is overturned, you get your money back.
  * If you are sanctioned again, it won’t be for a longer period.

####  Things to deal with first

We know that if you’ve been given a sanction, your priority is how to survive with little or no benefit.

[What to do if you have been given a sanction](/get-help/benefits/universal-credit-uc/challenge-universal-credit-sanction "What to do if you get a Universal Credit sanction") shows you what you need to do immediately (apply for a hardship loan, find a food bank, speak to your bank etc). Come back to this tool after you have followed those steps.
